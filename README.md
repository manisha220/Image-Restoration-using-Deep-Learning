# Image-Restoration-using-Deep-Learning

This repository contains my project **Image Restoration Using U-Net**, developed to restore noisy or degraded images using deep learning techniques.

The project uses a **U-Net based convolutional neural network architecture** to perform image reconstruction by learning how to remove noise and recover clearer image details.

## Technologies Used

* Python
* TensorFlow / Keras
* Deep Learning
* U-Net

## Project Features

* Restores noisy or degraded images using a U-Net model
* Performs image denoising and reconstruction
* Trained on image datasets for learning restoration patterns
* Produces clearer and enhanced output images

## Purpose

The main objective of this project is to demonstrate the application of deep learning in image restoration tasks, especially for improving image quality through automated noise removal.

## Note

This project highlights the use of convolutional neural networks for practical image enhancement and restoration applications.
